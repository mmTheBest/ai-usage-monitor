# AI Usage Monitor

A small native macOS desktop widget for local AI usage monitoring. It can show enabled subscription or API usage sections for supported providers such as Codex, Claude Code, OpenAI API, Anthropic API, Gemini API, DeepSeek API, and GLM API.

## Install

Requirements:

- macOS 14 or newer
- Xcode Command Line Tools
- Swift Package Manager

Run from source:

```bash
./Scripts/start-analytics-widget.command
```

Build a local app bundle:

```bash
./Scripts/build-release-app.command
```

The app bundle is written to:

```text
dist/AI Usage Monitor.app
```
